# XaeroPlus-Dependent Modules

## BetterStashFinder, TrailFollower

These modules require **XaeroPlus** and **Xaero's Minimap** to be installed at runtime.

### Installation Instructions

1. Download and install Xaero's Minimap from https://modrinth.com/mod/xaeros-minimap
2. Download and install XaeroPlus from https://modrinth.com/mod/xaeroplus  
3. Make sure both mods are for Minecraft 1.21.5
4. Place the mod files in your mods folder alongside Meteor Client

### What These Modules Do

**BetterStashFinder**: Enhanced version of StashFinder with:
- Old chunk detection (1.12 and 1.19+ chunks)
- Xaero's Minimap waypoint integration
- Discord webhook notifications
- Advanced logging options

**TrailFollower**: Automatically follows player trails:
- Detects old chunks from other players
- Uses Baritone for automatic pathfinding
- Nether and Overworld support
- Trail deviation detection

If these mods are not installed, the modules will show an error message when activated.
